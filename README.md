# Commit rules:

1. PR 형식 - 
  ex) [Client] / #88 /  edit: readme
  
2. Commit message 형식 - gitmoji
  [title] / body
  ex) edit: readme
  
  참고 자료: https://gitmoji.carloscuesta.me/

3. branch 이름 형식 -
  ex) feat / login
  ex) fix / readme

  참고 자료: https://hyeran-story.tistory.com/99

4. 변수 이름 -
  Camel-case
  ex) clientLogin

5. 파일 & 생성자 이름 -
  Pascal-case
  ex) ClientSide
  
6. node & npm 버전 통일 -
  Node v12.18.2
  NPM v6.14.5
